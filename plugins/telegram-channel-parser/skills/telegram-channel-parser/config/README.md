# Настройка telegram-channel-parser

## Быстрый старт

Скилл работает **без настройки**. Дефолтный набор каналов (AI/tech community) уже зашит в скрипт — дайджест работает из коробки.

## Кастомизация каналов

Чтобы использовать свой набор каналов:

```bash
cp config/.env.example config/.env
```

Отредактируйте `TG_CHANNELS` в `.env`.

### Приоритет каналов

1. `--channels "a,b,c"` — явный параметр (высший приоритет)
2. `TG_CHANNELS` из `config/.env` — пользовательский набор
3. Встроенный набор community-каналов — дефолт

### Параметры .env

| Переменная | Описание | По умолчанию |
|-----------|----------|--------------|
| `TG_CHANNELS` | Каналы для дайджеста (через запятую, без @) | community list |
| `TG_CACHE_TTL_HOURS` | Время жизни кеша в часах | `6` |

## Ограничения

- Только публичные каналы (с включённым t.me/s/ превью)
- Rate limit: 1.5с между запросами
- Пагинация: ~20 постов на страницу
